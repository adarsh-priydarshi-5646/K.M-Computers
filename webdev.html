Learn HTML and CSS

A comprehensive guide to building and styling web pages

What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages by defining elements like text, images, links, and forms.

Basic HTML Structure

Every HTML document begins with a doctype declaration, followed by the ``, ``, and `` tags. The `` contains meta-information like the title, while the `` holds the visible content.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first web page created using HTML.</p>
  </body>
</html>

Common HTML Tags

  • <h1> to <h6>: Headings, where <h1> is the largest and <h6> is the smallest.
  • <p>: Defines a paragraph of text.
  • <a href="URL">: Defines a hyperlink to another page.
  • <img src="image.jpg">: Embeds an image on the page.
  • <ul> and <ol>: Creates unordered and ordered lists respectively.
  • <form>: Defines a form for user input.
  • <div>: A container for other HTML elements, often used for layout purposes.
  • <span>: A generic inline container for text or other inline elements.

What is CSS?

CSS stands for Cascading Style Sheets. It is used to control the presentation (design) of a web page by applying styles like color, layout, fonts, and spacing.

CSS Syntax

CSS is written as a set of rules consisting of a selector and a declaration block. The selector targets HTML elements, while the declaration block contains the styling properties to apply.

h1 {
    color: blue;
    text-align: center;
}

p {
    font-size: 18px;
    color: gray;
}

Common CSS Properties

  • color: Sets the text color.
  • font-size: Defines the size of the font.
  • background-color: Sets the background color of an element.
  • margin: Adds space around an element.
  • padding: Adds space inside an element.
  • text-align: Aligns the text (left, center, right).
  • border: Adds a border around an element.
  • display: Controls the display behavior of an element (e.g., block, inline, flex).

Combining HTML and CSS

You can combine HTML and CSS by linking an external CSS file to the HTML document, embedding CSS in the `` section, or using inline CSS within an HTML tag.

Example of External CSS

<link rel="stylesheet" href="styles.css">
<h1>Welcome to My Website</h1>
<p>This page is styled using an external CSS file.</p>

Example of Inline CSS

<h1 style="color: green; text-align: center;">Hello, World!</h1>

Example of Internal CSS

<style>
  h1 {
    color: red;
    font-size: 2em;
  }
</style>
<h1>Hello, World!</h1>

Advanced HTML Tags

Forms

Forms allow users to input data. Below is an example of an HTML form:

<form action="submit_form.php" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">

  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <input type="submit" value="Submit">
</form>

Multimedia Tags

You can embed images, audio, and video in your web page:

<img src="image.jpg" alt="Description">
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>
<video controls>
  <source src="video.mp4" type="video/mp4">
</video>

HTML5 Semantic Elements

HTML5 introduced several new semantic elements to improve the structure and accessibility of web pages. These include:

  • <header>: Represents introductory content or navigation links.
  • <article>: Represents an independent piece of content.
  • <section>: Represents a section of content.
  • <footer>: Represents footer content.
  • <nav>: Represents navigation links.
  • <aside>: Represents content that is tangentially related to the main content.
  • <main>: Represents the main content of the document.

Advanced CSS: Flexbox and Grid Layouts

CSS Flexbox and Grid are powerful layout tools that allow you to create complex and responsive layouts easily.

Flexbox Example

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.item {
    width: 100px;
    height: 100px;
    background-color: #4CAF50;
    color: white;
    display: flex;
    justify-content: center;
    align-items: center;
}

CSS Grid Example

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

.item {
    background-color: #4CAF50;
    color: white;
    padding: 20px;
}
