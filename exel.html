Excel Mastery: All Functions, Formulas, and Techniques

Learn the complete range of Excel functions, formulas, and techniques to become an expert at data analysis, automation, and productivity!

Basic Excel Functions

  • SUM:
    =SUM(A1:A5)
    Adds values in the range A1 to A5.
  • AVERAGE:
    =AVERAGE(A1:A5)
    Calculates the average of the numbers in the range A1 to A5.
  • MIN/MAX:
    =MIN(A1:A5)
    Returns the smallest value in the range A1 to A5.
  • COUNT:
    =COUNT(A1:A5)
    Counts the number of numeric entries in the range A1 to A5.
  • IF:
    =IF(A1 > 10, "Yes", "No")
    Returns "Yes" if the value in A1 is greater than 10, otherwise "No".

Logical Functions

  • AND:
    =AND(A1 > 10, B1 < 5)
    Returns TRUE if both conditions are met.
  • OR:
    =OR(A1 > 10, B1 < 5)
    Returns TRUE if at least one condition is met.
  • NOT:
    =NOT(A1 > 10)
    Reverses the TRUE/FALSE result of a condition.

Lookup Functions

  • VLOOKUP:
    =VLOOKUP(A2, B2:D10, 3, FALSE)
    Looks for a value in the first column of a range and returns a value from a specified column.
  • HLOOKUP:
    =HLOOKUP(A2, B1:D10, 3, FALSE)
    Similar to VLOOKUP, but searches horizontally.
  • INDEX & MATCH:
    =INDEX(C2:C10, MATCH(E2, B2:B10, 0))
    More flexible than VLOOKUP; finds a value based on row/column lookup.

Text Functions

  • CONCATENATE:
    =CONCATENATE(A1, " ", B1)
    Joins the values in cells A1 and B1 with a space in between.
  • LEFT/RIGHT:
    =LEFT(A1, 5)
    Extracts the leftmost 5 characters from A1.
  • MID:
    =MID(A1, 3, 5)
    Extracts 5 characters from the middle of the text in A1, starting from the 3rd character.
  • TRIM:
    =TRIM(A1)
    Removes extra spaces from the text in A1.

Date and Time Functions

  • NOW:
    =NOW()
    Returns the current date and time.
  • DATE:
    =DATE(2024, 12, 31)
    Returns a date from the given year, month, and day.
  • YEAR/MONTH/DAY:
    =YEAR(A1), =MONTH(A1), =DAY(A1)
    Extracts the year, month, or day from a date in A1.

Advanced Excel Functions

  • SUMIFS:
    =SUMIFS(C2:C10, B2:B10, "Product A", A2:A10, ">10")
    Sums the values in C2:C10 where the conditions are met.
  • COUNTIFS:
    =COUNTIFS(A2:A10, ">10", B2:B10, "Product A")
    Counts the values based on multiple conditions.
  • OFFSET:
    =OFFSET(A1, 3, 2)
    Returns the value of a cell offset from another.
  • INDIRECT:
    =INDIRECT("A1")
    Returns the value of a cell referenced by a string.

Excel Automation: Macros and VBA

Automate repetitive tasks in Excel by recording macros or writing VBA code. This can save significant time and effort.

Example Macro:

Sub HighlightCells()
Dim cell As Range
For Each cell In Range("A1:A10")
If cell.Value > 10 Then
cell.Interior.Color = RGB(255, 0, 0)
End If
Next cell
End Sub
